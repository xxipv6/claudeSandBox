# /deploy - 部署命令

## 执行流程

1. **部署前检查**
   - 代码是否已提交
   - 测试是否通过
   - 文档是否更新

2. **选择环境**
   - 开发环境
   - 测试环境
   - 预生产环境
   - 生产环境

3. **执行部署**
   - 调用 ops agent
   - 部署配置
   - 执行部署

4. **验证部署**
   - 健康检查
   - 功能验证
   - 性能检查

5. **部署后监控**
   - 错误日志
   - 性能指标
   - 用户反馈

## 使用场景

- ✅ 部署新版本
- ✅ 回滚版本
- ✅ 紧急修复部署
- ✅ 配置更新
