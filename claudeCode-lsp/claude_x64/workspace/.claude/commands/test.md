# /test - 测试命令

## 执行流程

1. **选择测试类型**
   - 单元测试
   - 集成测试
   - 端到端测试
   - 全部测试

2. **运行测试**
   - 自动检测测试框架
   - 执行测试套件
   - 收集结果

3. **检查覆盖率**
   - 计算代码覆盖率
   - 检查是否达到 80%
   - 生成覆盖率报告

4. **分析结果**
   - 失败测试分析
   - 覆盖率缺口分析
   - 改进建议

## 使用场景

- ✅ 开发完成后验证
- ✅ 提交前检查
- ✅ CI/CD 流水线
- ✅ 覆盖率报告
