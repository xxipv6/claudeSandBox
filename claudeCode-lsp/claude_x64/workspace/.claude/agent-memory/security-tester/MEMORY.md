# Security-Tester Agent Memory

## 项目知识引用

本项目使用**共享知识库**，所有 knowledge 文件夹的内容对 SECURITY-TESTER 强制可见：

### 共享知识库位置
- **失败模式库**：`/workspace/knowledge/patterns.md`
- **领域知识库**：`/workspace/knowledge/domains.md`
- **工具认知库**：`/workspace/knowledge/tools.md`
- **错误学习库**：`/workspace/knowledge/corrections.md`

### Security-Tester 使用指引

**分析前**：
- 快速浏览 patterns.md 的所有模式（特别是安全研究特定模式）
- 参考 tools.md 选择合适的工具组合

**分析中**：
- 重点关注：攻击路径、权限绕过、业务逻辑漏洞
- 对照 corrections.md 避免重复错误

**分析后**：
- 检查是否遗漏了常见的攻击路径
- 将新发现的模式记录到 corrections.md

## 分析重点

### 威胁建模
- 攻击面有哪些？（接口、数据、逻辑、状态）
- 攻击者可能的目标是什么？
- 攻击者有什么能力？
- 攻击路径是什么？

### 权限与授权分析
- 是否存在越权访问？（水平越权、垂直越权）
- 是否存在未授权操作？
- 权限检查是否完整？
- 是否存在权限检查绕过？

### 状态机与业务逻辑分析
- 是否存在状态跳跃或非法状态转换？
- 是否存在业务规则绕过？
- 是否存在竞态条件或时序漏洞？
- 是否存在滥用正常功能的路径？

### 输入与接口安全分析
- 是否存在未校验或校验宽松的输入？
- 是否存在参数污染或注入风险？
- 是否存在接口滥用？
- 是否存在批量操作的滥用？

### 信息泄露分析
- 是否暴露敏感信息？
- 错误信息是否泄露系统细节？
- 是否存在调试信息暴露？
- 是否存在可被枚举的信息？

---

## 常见安全漏洞

### 权限问题
- 水平越权：用户 A 访问用户 B 的资源
- 垂直越权：普通用户获得管理员权限
- 跨租户越权：租户 A 访问租户 B 的资源

### 业务逻辑漏洞
- 价格操纵
- 优惠券滥用
- 积分/限额绕过
- 工作流 bypass

### 状态机绕过
- 状态回滚
- 状态跳跃
- 并发状态转换
- 状态不一致利用

### 信息泄露
- 错误消息泄露
- 调试信息暴露
- 枚举攻击
- 时间侧信道

## 输出要求

必须输出结构化分析，包括：
- 威胁建模
- 漏洞分析
- 攻击路径图
- 风险汇总
- 修复建议
