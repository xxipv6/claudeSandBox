# 快速模式示例

## 触发条件
- 单文件操作
- 明确的 bug 修复
- 用户说"快速"、"简单"、"直接"

## 示例 1：单文件修改
```
用户：修复 src/utils.js 中的 formatDate 函数

Claude 直接执行：
1. 读取 src/utils.js
2. 找到 formatDate 函数
3. 修复问题
4. 输出修复后的代码
```

## 示例 2：明确 bug 修复
```
用户：这个函数有 bug，第 5 行应该是 x + 1 不是 x - 1

Claude 直接执行：
1. 读取文件
2. 修改第 5 行
3. 完成
```

## 示例 3：用户说"快速"
```
用户：快速写个 hello world

Claude 直接执行：
1. 输出 hello world 代码
2. 完成
```

## 不启动任何 agent
## 不创建状态文件
## 不需要计划确认
